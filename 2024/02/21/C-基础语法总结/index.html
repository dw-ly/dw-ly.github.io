注意事项

  1. 内存管理：

    • 使用智能指针代替裸指针
    • 注意配对使用 new/delete 或 malloc/free
    • 防止内存泄漏

  2. 常量使用：

    • 优先使用 const 确保不可修改性
    • 考虑使用 constexpr 进行编译期优化

  3. 命名空间：

    • 避免使用 using namespace
    • 优先使用具体的 using 声明

  4. 错误处理：

    • 合理使用异常处理
    • 注意资源的正确释放
